        PROTOTYPE:
        ---------
        a prototype is an object which is associated with every object and function in javascript.and the object's prototype is not visible.

        PROTOTYPE CHAINING :
        --------------------

        in javascript ,functions and arrays are objects.
        an every object has a prototype. prototype is a default property of every object and it holds another object 
        That prototype object has a prototype of its own, and so on until an object is reached with null as its prototype.

        CREATION PHASE AND EXECUTION PHASE :
        ----------------------------------
        Javascript is a synchronous programming language.
        Creation Phase : it runs two times. in the first time, it stores the function declarations in the memory.
                        in the second run , assigns undefined to all the variables.
        Execution Phase : in this phase, all the variables are assigned with values and the functions were executed.

        CALLBACK:
        ---------
        A callback is a function passed as an argument to another function.

        -----------
